# JavaScript - Number Methods (دوال الأرقام)

الجافاسكريبت بتوفرلنا دوال جاهزة (`Built-in Methods`) للتعامل مع الأرقام، سواء عشان نحولها لنصوص، أو نستخرجها من نصوص، أو نتحقق من صحتها.

---

## 1. The Syntax Trap (فخ طريقة الكتابة)

في الجافاسكريبت، لو كتبت رقم صحيح وحطيت بعده نقطة عشان تستدعي دالة (زي `100.toString()`)، المتصفح هيفتكر إن النقطة دي علامة عشرية (`Decimal Point`) والكود هيضرب `Error`.
عشان كده عندنا طريقتين صح:
- نحط الرقم بين أقواس `(100).toString()`
- أو نحط نقطتين ورا بعض `100..toString()` (الأولى للعشرية والثانية للدالة).

---

## 2. Formatting & Conversion (التحويل والتنسيق)

- **دالة (`toString`):** بتحول الرقم العادي إلى نص (`String`).
- **دالة (`toFixed`):** بتتحكم في عدد الأرقام العشرية اللي بتظهر بعد العلامة، وبتقرب الرقم لأقرب قيمة، و**بترجع الناتج في هيئة نص**.

```javascript
// التحويل لنص
console.log((100).toString()); // "100"
console.log(100.10.toString()); // "100.1"

// التقريب وتحديد عدد الأرقام العشرية
console.log(100.554555.toFixed(2)); // "100.55" (رجع كنص)
```

---

## 3. Parsing (استخراج الأرقام من النصوص)

هنا بيبان الفرق الجوهري بين محاولة تحويل النص بالكامل لرقم، وبين "استخراج" الرقم من النص.

> [!danger] Number vs parseInt
> استخدام `Number()` أو علامة الزائد `+` بيجبر الجافاسكريبت تقرأ النص كله، فلو فيه حروف هترجع `NaN` (Not a Number). لكن `parseInt` بتمشي من الشمال لليمين، أول ما تلاقي رقم بتاخده، وأول ما تخبط في حرف بتقف وتتجاهل الباقي. (لو النص بدأ بحرف، هترجع `NaN`).

- **دالة (`parseInt`):** بتستخرج الرقم الصحيح فقط (بدون كسور).
- **دالة (`parseFloat`):** بتستخرج الرقم بالكسور العشرية بتاعته.

```javascript
// 1. التحويل الصارم (فشل بسبب وجود حروف)
console.log(Number("100 Osama")); // NaN
console.log(+"100 Osama");        // NaN

// 2. الاستخراج الذكي للأرقام الصحيحة
console.log(parseInt("100 Osama"));     // 100 (تجاهل كلمة Osama)
console.log(parseInt("Osama 100 Osama")); // NaN (فشل لأن النص بدأ بحرف)
console.log(parseInt("100.500 Osama")); // 100 (تجاهل الكسور والحروف)

// 3. الاستخراج الذكي بالأرقام العشرية
console.log(parseFloat("100.500 Osama")); // 100.5 (استخرج الرقم بالكسر)
```

## 4. Verification Methods (دوال التحقق - ES 6)

دي دوال حديثة نزلت في ES 6 عشان نتأكد من نوع وقيمة الرقم قبل ما نعمل عليه عمليات حسابية.

- **دالة (`Number.isInteger`):** بتفحص هل القيمة اللي معاك رقم صحيح ولا لأ (بترجع `true` أو `false`).
- **دالة (`Number.isNaN`):** بتفحص هل العملية الحسابية باظت وطلعت `NaN` ولا لأ.

```javascript
// التحقق من الأرقام الصحيحة
console.log(Number.isInteger("100"));   // false (لأنه نص مش رقم)
console.log(Number.isInteger(100.500)); // false (لأنه رقم عشري مش صحيح)
console.log(Number.isInteger(100));     // true  (رقم صحيح)

// التحقق من الأخطاء الحسابية
// قسمة نص على رقم بتنتج NaN
console.log(Number.isNaN("Osama" / 20)); // true (بالفعل النتيجة NaN)
```
