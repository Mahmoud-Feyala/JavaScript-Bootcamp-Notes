# BOM - Open & Close Windows (فتح وإغلاق النوافذ)

الجزء ده مخصص للتحكم في النوافذ المنبثقة وإنشاء التابات الجديدة برمجياً.

---

## 1. Open Method (دالة الفتح)

دالة `window.open` بتقبل 4 معاملات (`Parameters`) كلهم اختياريين، ولو مكتبتش فيهم حاجة المتصفح هيفتح تاب جديد فاضي:

- **الرابط (`URL`):** مسار الصفحة اللي عايز تفتحها. لو سبته فاضي هيفتح صفحة بيضاء (`about:blank`).
- **الهدف (`Target`):** مكان الفتح. هل يفتح في تاب جديد (`_blank`) ولا في نفس الصفحة (`_self`) ولا في نافذة ليها اسم معين.
- **الخصائص (`Window Features`):** نص (`String`) بنحدد فيه مقاسات وشكل النافذة (مفصولين بفاصلة بدون مسافات).
- **الاستبدال (`History Replace`):** قيمة منطقية (`Boolean`) بتحدد هل الرابط الجديد يستبدل القديم في تاريخ المتصفح ولا لأ.

---

## 2. Window Features (خصائص النافذة)

لما بنكتب الخصائص دي، المتصفح بيفتح الصفحة في نافذة منبثقة (`Popup`) منفصلة مش مجرد تاب جديد.

- **العرض والطول (`width` & `height`):** مقاس النافذة بالبيكسل.
- **المسافة من اليسار والأعلى (`left` & `top`):** مكان النافذة على شاشة اليوزر بالبيكسل (بتبدأ من فوق على الشمال).
- **شريط القوائم (`menubar`):** إظهار أو إخفاء شريط القوائم (بياخد `yes` أو `no`).

> [!info] Search Task (تحديثات أمنية)
> نقطة البحث الخاصة بخصائص النافذة مهمة جداً. المتصفحات الحديثة زي `Chrome` بقت بتمنع أو تتجاهل خصائص كتير (زي إخفاء شريط العنوان أو تعطيل التمرير) كنوع من الحماية لليوزر (`Security Reasons`) عشان المواقع متعملش نوافذ خبيثة ميعرفش يقفلها.

---

## 3. Close Method (دالة الإغلاق)

دالة `window.close()` بتقفل النافذة المفتوحة.

> [!danger] Security Trap (فخ الأمان)
> الجافاسكريبت تقدر تقفل **فقط** النوافذ اللي هي فتحتها بنفسها بـ `window.open()`. لو حاولت تستخدم الكود ده عشان تقفل تاب اليوزر فتحه بإيده، المتصفح هيرفض وهيطلعلك `Warning` في الكونسول إن الـ Script ملوش صلاحية يقفل النافذة دي.

```javascript
// 1. فتح نافذة في نفس التاب بعد ثانيتين
setTimeout(function () {
  window.open("", "_self", "", false);
}, 2000);

// 2. فتح موقع جوجل في نافذة منبثقة بمقاسات ومكان محدد
setTimeout(function () {
  window.open(
    "[https://google.com](https://google.com)",
    "_blank",
    "width=400,height=400,left=200,top=10"
  );
}, 2000);

// 3. طريقة الإغلاق (بتتطبق غالباً على النافذة المنبثقة)
// بنحفظ النافذة في متغير الأول عشان نقدر نتحكم فيها بعدين
// let myWindow = window.open("[https://google.com](https://google.com)", "_blank", "width=400,height=400");
// myWindow.close();
```
