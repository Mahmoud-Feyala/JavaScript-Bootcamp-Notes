# ES 6 - Destructuring Arrays (تفكيك المصفوفات)

الـ `Destructuring` عبارة عن تعبير (`Expression`) في الجافاسكريبت بيسمح لنا نستخرج بيانات من المصفوفات (`Arrays`) أو الكائنات (`Objects`) أو الـ `Maps` ، ونوزعها على متغيرات جديدة منفصلة في سطر واحد بس.



---

## 1. Basic Extraction & Default Values (الاستخراج الأساسي والقيم الافتراضية)

بدل ما تنادي على كل عنصر بالـ `Index` بتاعه، بتعمل مصفوفة وهمية فيها أسماء المتغيرات اللي عايزها، والجافاسكريبت بتوزع القيم بالترتيب.

> [!info] Default Values (القيم الافتراضية)
> تقدر تدي المتغير قيمة افتراضية. لو المصفوفة كان فيها قيمة للمتغير ده، هياخدها. لو مفيش (أو كانت `undefined`)، هياخد القيمة الافتراضية اللي إنت محددها عشان الكود ميضربش.

```javascript
let a = 1;
let b = 2;
let c = 3;
let d = 4;

let myFriends = ["Ahmed", "Sayed", "Ali", "Maysa"];

// 1. إعادة تعيين قيمة a لـ "A" (لو مفيش قيمة في المصفوفة)
// 2. توزيع باقي القيم بالترتيب
// 3. إعطاء e قيمة افتراضية "Osama" لأن المصفوفة الأصلية مفيهاش عنصر خامس
[a = "A", b, c, d, e = "Osama"] = myFriends;

console.log(a); // Ahmed (أخدت القيمة من المصفوفة وتجاهلت القيمة الافتراضية)
console.log(b); // Sayed
console.log(e); // Osama (أخدت القيمة الافتراضية لأن المصفوفة خلصت)
```

---

## 2. Skipping Elements (تخطي العناصر)

لو إنت محتاج العنصر التاني والرابع بس، مش مضطر تعمل متغيرات للعنصر الأول والتالت وتملى بيهم الذاكرة. مجرد وضع فاصلة (`,`) فاضية بيخلي الجافاسكريبت تتجاهل العنصر اللي في الترتيب ده.

```javascript
let myFriends = ["Ahmed", "Sayed", "Ali", "Maysa"];

// تجاهل العنصر الأول والتالت بوضع فواصل فارغة
let [, y, , z] = myFriends;

console.log(y); // Sayed (تجاهل Ahmed)
console.log(z); // Maysa (تجاهل Ali)
```

## 3. Advanced Destructuring (التفكيك المتداخل)



لو المصفوفة بتاعتك معقدة وجواها مصفوفات تانية (`Nested Arrays`)، كل اللي عليك تعمله إنك ترسم نفس الهيكل المعقد ده بالظبط في ناحية المتغيرات.

> [!danger] Logic Breakdown 
> المصفوفة الأصلية فيها 4 عناصر أساسية، العنصر الرابع عبارة عن مصفوفة جواها مصفوفة. عشان نوصل لـ `Shady` و `Gamal`:
> 1. حطينا 3 فواصل `, , ,` عشان نتجاهل أول 3 أصدقاء.
> 2. فتحنا أقواس مصفوفة جديدة `[]` عشان ندخل جوه المصفوفة الرابعة.
> 3. مسكنا أول عنصر `a` (اللي هو `Shady`).
> 4. حطينا فاصلة عشان نتجاهل `Amr`.
> 5. فتحنا أقواس جديدة `[]` عشان ندخل للمصفوفة الأخيرة.
> 6. حطينا فاصلة عشان نتجاهل `Mohamed` ، وبعدها مسكنا `b` (اللي هو `Gamal`).

```javascript
let myFriends = ["Ahmed", "Sayed", "Ali", ["Shady", "Amr", ["Mohamed", "Gamal"]]];

// التطابق الهيكلي خطوة بخطوة
let [, , , [a, , [, b]]] = myFriends;

console.log(a); // Shady
console.log(b); // Gamal
```

# ES 6 - Swapping Variables (تبديل المتغيرات)

عملية تبديل القيم بين متغيرين (`Swapping`) كانت بتحتاج دايماً متغير تالت كـ "وسيط" عشان نحفظ فيه القيمة مؤقتاً قبل ما تضيع. مع ظهور الـ `Destructuring` في `ES6` ، العملية دي بقت بتتم في سطر واحد وبدون أي متغيرات إضافية.

---
## 1. The Old Way (الطريقة القديمة - المتغير الوسيط)

في الطريقة الكلاسيكية، كنا بنضطر نكريت متغير جديد (غالباً بنسميه `temp` أو `stash`) عشان نشيل فيه قيمة المتغير الأول، عشان لما نغيره بقيمة المتغير التاني منخسرش البيانات.

```javascript
let book = "Video";
let video = "Book";

// 1. حفظ قيمة book في متغير مؤقت
let stash = book; // stash دلوقتي قيمته "Video"

// 2. تغيير قيمة book بقيمة video
book = video; // book دلوقتي قيمته "Book"

// 3. تغيير قيمة video بالقيمة اللي حفظناها في المتغير المؤقت
video = stash; // video دلوقتي قيمته "Video"
```

---

## 2. The ES 6 Way (الطريقة الحديثة - تفكيك المصفوفات)

بنستخدم فكرة تفكيك المصفوفات عشان نعمل التبديل. بنحط المتغيرين اللي عايزين نغير قيمهم في مصفوفة على الشمال، وبنحط المتغيرين بقيمهم الجديدة في مصفوفة على اليمين بالترتيب المعكوس.

> [!info] How It Works
> الجافاسكريبت هنا بتعمل مصفوفة مؤقتة في الذاكرة (خلف الكواليس) فيها القيمتين `["Book", "Video"]` ، وبعدين بتفكك المصفوفة دي وتوزع العنصر الأول على المتغير `book` والعنصر التاني على المتغير `video` في نفس اللحظة.

```javascript
let book = "Video";
let video = "Book";

// سطر واحد بيقوم بكل المهمة
[book, video] = [video, book];

console.log(book);  // "Book"
console.log(video); // "Video"
```
