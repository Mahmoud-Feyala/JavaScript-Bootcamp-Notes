# BOM - Window Methods (دوال النافذة والتمرير)

الجزء ده بيكمل تحكمنا في الـ `Window` وبيوفرلنا دوال سريعة للتحكم في حالة الصفحة (زي الطباعة والإيقاف) ودوال تانية للتحكم في الـ `Scroll` (التمرير) بدقة عالية.

---

## 1. General Methods (دوال عامة للنافذة)

- **الطباعة (`print`):** بتفتح شاشة الطباعة الخاصة بالمتصفح، كأن اليوزر ضغط `Ctrl + P`. (ممتازة جداً في صفحات الفواتير أو التقارير).
- **إيقاف التحميل (`stop`):** بتوقف تحميل الصفحة فوراً، كأن اليوزر ضغط على علامة الـ `X` اللي جنب زرار الـ `Refresh` في المتصفح.
- **التركيز (`focus`):** بتخلي النافذة هي النشطة وتجيبها في المقدمة. (مفيدة لو فاتح كذا نافذة منبثقة بـ `window.open` وعايز ترجع التركيز لواحدة معينة).

```javascript
// فتح نافذة جديدة
// let myNewWindow = window.open("[https://google.com](https://google.com)", "", "width=500,height=500");

// إعطاء التركيز للنافذة الجديدة عشان تظهر فوق كل النوافذ
// myNewWindow.focus();
```

---

## 2. Scrolling Methods (دوال التمرير)

الفرق بين الدوال دي هو اللي بيحدد الصفحة هتتحرك إزاي. كلهم بياخدوا إحداثيات العرض `X` (لليمين واليسار) والطول `Y` (لفوق ولتحت).

> [!danger] Interview Trap: scrollTo vs scrollBy (فخ الانترفيوهات)
> - **التمرير المطلق (`scrollTo` أو `scroll`):** الاتنين بيعملوا نفس الحاجة بالظبط. بيروحوا لنقطة **محددة وثابتة** في الصفحة (مثلاً: انزل للبيكسل رقم 500). لو نفذت الدالة تاني الصفحة مش هتتحرك لأنها وصلت للنقطة دي خلاص.
> - **التمرير النسبي (`scrollBy`):** بيتحرك بمسافة **إضافية** بناءً على مكانه الحالي (مثلاً: انزل 500 بيكسل زيادة من مكانك دلوقتي). لو نفذتها تاني هينزل 500 بيكسل كمان.

---

## 3. The Options Object (كائن الخيارات)

بدل ما تبعت الأرقام `X` و `Y` مباشرة والطرحة تنط فجأة، الأفضل والحديث إنك تبعت كائن (`Object`) جواه الخيارات دي، عشان تقدر تتحكم في نعومة التمرير (`Smooth Scrolling`).

```javascript
// 1. التمرير العادي السريع لمكان محدد
// window.scrollTo(500, 200); 

// 2. التمرير الناعم (الطريقة الاحترافية الحديثة)
window.scrollTo({
  left: 500, // يمثل محور X
  top: 200,  // يمثل محور Y
  behavior: "smooth" // التمرير هيكون ناعم مش قفزة مفاجئة
});

// 3. التمرير المتكرر (scrollBy)
// كل مرة الكود ده هيتنفذ، الصفحة هتنزل لتحت مسافة إضافية
/*
window.scrollBy({
  top: 100,
  behavior: "smooth"
});
*/
```
