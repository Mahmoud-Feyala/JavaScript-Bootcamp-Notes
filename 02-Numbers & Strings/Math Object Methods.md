# JavaScript - Math Object (كائن الرياضيات)

كائن `Math` بيوفرلنا مجموعة من الدوال الثابتة للقيام بالعمليات الحسابية المعقدة والتقريب. كل الدوال دي بنستدعيها بكتابة `Math.` قبل اسم الدالة.



---

## 1. Rounding Methods (دوال التقريب)

دي أهم مجموعة دوال، وكل واحدة فيهم ليها سلوك مختلف في التعامل مع الكسور العشرية.

- **دالة (`round`):** التقريب الرياضي الطبيعي. لو الكسر `0.5` أو أكبر بيقرب للرقم الأعلى، ولو أقل بيقرب للرقم الأقل.
- **دالة (`ceil`):** (السقف) دايماً بتقرب للرقم الصحيح الأعلى، حتى لو الكسر `0.1`.
- **دالة (`floor`):** (الأرضية) دايماً بتقرب للرقم الصحيح الأقل، وبترمي الكسر تماماً.
- **دالة (`trunc`):** (القص - ES 6) مبتعملش أي تقريب، هي بس بتقص الكسر العشري أياً كان وترجع الرقم الصحيح.

```javascript
// التقريب الطبيعي
console.log(Math.round(99.2)); // 99
console.log(Math.round(99.5)); // 100

// السقف (التقريب للأعلى دائماً)
console.log(Math.ceil(99.2));  // 100

// الأرضية (التقريب للأسفل دائماً)
console.log(Math.floor(99.9)); // 99

// قص الكسر العشري
console.log(Math.trunc(99.5)); // 99
```

## 2. Min & Max Methods (دوال القيمة الصغرى والكبرى)

بتستخدم لاستخراج أكبر أو أصغر رقم من وسط مجموعة أرقام. الدالة دي بتقبل الأرقام مفصولة بفاصلة (مبتقبلش مصفوفة بشكل مباشر بدون استخدام الـ `Spread Operator`).

```javascript
// استخراج أصغر قيمة
console.log(Math.min(10, 20, 100, -100, 90)); // -100

// استخراج أكبر قيمة
console.log(Math.max(10, 20, 100, -100, 90)); // 100
```

## 3. Power & Random Methods (دوال الأس والعشوائية)

- **دالة (`pow`):** اختصار لـ Power، وبتستخدم لرفع رقم لأس معين (الرقم الأساسي، ثم الأس).
- **دالة (`random`):** بتولد رقم عشري عشوائي بين الصفر (ممكن يكون صفر) والواحد (مستحيل يوصل لواحد). دايماً بنضربها في رقم ونستخدم معاها `Math.trunc` أو `Math.floor` عشان نطلع أرقام صحيحة عشوائية.

```javascript
// 2 أس 4
console.log(Math.pow(2, 4)); // 16

// توليد رقم عشوائي
console.log(Math.random()); // مثال للناتج: 0.84729482
```
