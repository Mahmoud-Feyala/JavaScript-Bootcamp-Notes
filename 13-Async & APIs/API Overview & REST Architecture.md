# JavaScript - API Overview And Tools

عشان نفهم فايدة الـ `JSON` الحقيقية، لازم نفهم إيه هو الـ `API`. الجيسون هو مجرد "لغة الحوار" أو الصيغة اللي بتتنقل بيها البيانات، لكن الـ `API` هو "الطريق" اللي البيانات دي بتمشي فيه بين السيرفر والـ Client.



---

## 1. API Overview 

- **واجهة برمجة التطبيقات** (`Application Programming Interface`): هو اختصار لـ API. عبارة عن كود بيسمح لتطبيقين إنهم يكلموا بعض ويتبادلوا البيانات.
- **مثال المطعم** (`Restaurant Example`): تخيل إنك (الـ `Client`) قاعد في مطعم وعايز تطلب أكل من المطبخ (الـ `Server`). الـ `API` هو "الجرسون" اللي بياخد طلبك يوديه المطبخ، ويرجعلك بالأكل (البيانات) في شكل منظم وهو الـ `JSON`.

---

## 2. Tools To Test API 

قبل ما تبدأ تكتب كود جافاسكريبت عشان تسحب بيانات من أي `API` ، لازم تختبر الرابط ده وتتأكد إن البيانات راجعة سليمة وشكلها إيه.

- **برامج متخصصة** (`Standalone Software`): زي برنامج `Postman` الشهير جداً. ده بيسمحلك تبعت طلبات معقدة للسيرفر وتختبر الـ `REST API` باحترافية وتشوف الـ `JSON` اللي راجع.
- **إضافات المتصفح** (`Browser Extensions`): زي إضافة `JSON Viewer` ، دي بتخلي شكل بيانات الجيسون اللي بتفتحها في المتصفح ملونة ومترتبة وممكن تقفل وتفتح الأقواس بتاعتها بسهولة بدل ما تظهر كنص واحد سايح على بعضه.

---

## 3. Preview Github API 

موقع `Github` بيوفر `API` مجاني ومفتوح نقدر نتدرب عليه. لما بتدخل على الروابط دي من المتصفح (أو من Postman)، السيرفر بيرد عليك بملف `JSON` كامل فيه بيانات حقيقية.

- **رابط المستخدمين** (`Users Endpoint`):
  `https://api.github.com/users/elzerowebschool`
  *(الرابط ده بيرجع كائن جيسون فيه بيانات حساب الزيرو زي الاسم، الصورة، عدد المتابعين).*

- **رابط المستودعات** (`Repositories Endpoint`):
  `https://api.github.com/users/elzerowebschool/repos`
  *(الرابط ده بيرجع مصفوفة جيسون مليانة كائنات، كل كائن بيمثل مشروع أو Repo موجود على الحساب).*
