##  What is this and why does it exist?

Most JavaScript tutorials focus only on syntax and quickly jumping into frameworks.

This repository documents my journey learning JavaScript step by step while understanding how the language actually works behind the scenes.

---

##  Philosophy

This repository follows a learning-by-understanding approach.

That means:

- Understanding concepts before memorizing syntax
- Writing code manually instead of copy-pasting
- Practicing challenges consistently
- Learning debugging and problem solving
- Building projects during the learning process
- Avoiding tutorial dependency
