# JavaScript - Date And Time Introduction

التعامل مع التاريخ والوقت في الجافاسكريبت بيعتمد بشكل أساسي على كائن مدمج في اللغة اسمه `Date`. المتصفح بيقرأ الوقت والتاريخ من نظام التشغيل بتاعك.

---
## 1. Date Constructor (مُنشئ التاريخ)

عشان تجيب الوقت والتاريخ الحاليين بالظبط لحظة تشغيل الكود، بنستخدم الكلمة المفتاحية `new` مع القالب الجاهز `Date()`.

```javascript
// استخراج الوقت والتاريخ الحالي
let dateNow = new Date();
console.log(dateNow); 
// النتيجة هتطبع اليوم، الشهر، السنة، والساعة بالظبط مع التوقيت المحلي
```

## 2. Epoch Time & Date.now (نقطة البداية ووقت اليونكس)

عشان الكمبيوتر يقدر يحسب الوقت ويسجله، كان لازم يتفقوا على **نقطة بداية الصفر** (`Starting Point`). في علوم الحاسب، النقطة دي هي يوم **1 يناير 1970**، والوقت اللي عدى من ساعتها لحد اللحظة دي بيتسمى وقت اليونكس (`Unix Time`).

دالة `Date.now()` بترجعلك الوقت اللي عدى من سنة 1970 لحد اللحظة الحالية بوحدة **الميللي ثانية** (`Milliseconds`). (الثانية الواحدة = 1000 ميللي ثانية).

```javascript
// الوقت الحالي بالميللي ثانية
console.log(Date.now()); 

// تحويل الميللي ثانية لوحدات أكبر (عمليات حسابية متتالية)
let seconds = Date.now() / 1000; // عدد الثواني
console.log(`Seconds: ${seconds}`);

let minutes = seconds / 60; // عدد الدقائق
console.log(`Minutes: ${minutes}`);

let hours = minutes / 60; // عدد الساعات
console.log(`Hours: ${hours}`);

let days = hours / 24; // عدد الأيام
console.log(`Days: ${days}`);

let years = days / 365; // عدد السنين
console.log(`Years: ${years}`); // هيطلعلك تقريباً عدد السنين اللي عدت من 1970 لحد دلوقتي
```
