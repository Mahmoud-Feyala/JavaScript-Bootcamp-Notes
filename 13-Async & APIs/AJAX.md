# JavaScript - AJAX Introduction 

تقنية الـ `AJAX` هي النقلة النوعية اللي خلت الويب سريع وتفاعلي. زمان، أي طلب من السيرفر كان بيحتاج إن الصفحة كلها تعمل `Refresh` (إعادة تحميل). لكن مع الأجاكس، تقدر تبعت وتستقبل بيانات في الخلفية من غير ما اليوزر يحس بأي تقطيع.

---

## 1. What is AJAX? 

- **الاختصار**: `Asynchronous JavaScript And XML` (جافاسكريبت و إكس إم إل اللامتزامنة). 
- **المفهوم**: الأجاكس مش لغة برمجة جديدة، هو "نهج" أو طريقة بنستخدم فيها تقنيات موجودة بالفعل مع بعض (HTML, CSS, JS, DOM) عشان نعمل طلبات للسيرفر في الخلفية.
- **ملاحظة هامة**: رغم إن اسمه فيه `XML` ، إلا إننا حالياً بنستخدم صيغة الـ `JSON` بنسبة 99% في نقل البيانات لأنه أسرع وأخف بكتير.

---

## 2. Real Life Examples 

عشان تحس بقيمة الأجاكس، دي أمثلة بنشوفها وبنستخدمها كل يوم:
- **YouTube Studio**: لما بترفع فيديو وبتعدل العنوان، التعديل بيتحفظ أوتوماتيك من غير ما الصفحة تحمل.
- **Google Drive**: بيعمل حفظ تلقائي (`Auto-save`) لملفاتك وإنت بتكتب.
- **Form Check Name**: لما تيجي تسجل حساب جديد وتكتب اسم مستخدم، وتلاقي الموقع بيقولك فوراً في نفس اللحظة "الاسم ده مستخدم قبل كده" من غير ما تدوس Submit.
- **Upload Photo**: رفع الصور وتغيير صورة البروفايل في الخلفية.

---

## 3. XMLHttpRequest Object 

عشان الجافاسكريبت تقدر تعمل سحر الأجاكس ده، بتستخدم كائن مبني جواها (في الـ Web API) اسمه `XMLHttpRequest`. الكائن ده هو "المندوب" بتاعك اللي بياخد الطلب يوديه للسيرفر، ويستنى الرد، ويرجعلك بالبيانات.
```javascript
// إنشاء نسخة جديدة من كائن الطلبات عشان نبدأ نكلم السيرفر
let req = new XMLHttpRequest();

// لو طبعنا الكائن ده في الكونسول، هنلاقي جواه خصائص كتير جداً هنتعامل معاها الدروس الجاية زي:
// - readyState (حالة الطلب الحالية)
// - status (كود الحالة زي 200 أو 404)
// - responseText (البيانات اللي راجعة من السيرفر)
console.log(req);
```
